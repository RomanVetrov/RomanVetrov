## 🧠 What I Build

```
📦 REST/gRPC services       →  Go + net/http/Gin/Chi + OpenAPI contracts
⚙️ concurrent workers       →  goroutines, channels, contexts, worker pools
📨 message-driven systems   →  Kafka / RabbitMQ / NATS consumers and producers
🗄️ data-heavy backends      →  PostgreSQL + MongoDB + Redis + migrations
🔐 auth & configuration     →  JWT/OAuth2, environment config, dependency wiring
🔁 reliable workflows       →  transactions, retries, idempotency, background jobs
📊 observable services      →  metrics, structured logs, tracing, dashboards
🧪 tested backend code      →  unit, integration, handler and race-aware tests
🐳 production delivery      →  Docker, compose, CI pipelines and Linux runtime
```

---

## 🎯 Engineering Focus

```
  Clear APIs              →  readable contracts, predictable handlers, simple clients
  Maintainable code       →  small packages, explicit dependencies, practical boundaries
  Reliable data flows     →  migrations, transactions, cache consistency, safe retries
  Error handling          →  explicit errors, useful logs, observable failure paths
  Practical testing       →  fast unit tests plus integration coverage where it matters
  Production awareness    →  configuration, metrics, health checks and deployment basics
```
